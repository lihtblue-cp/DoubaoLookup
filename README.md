必要条件：                                                                                                                                                                                            
  - macOS 13+ (Ventura)                                                                                                                                                                                 
  - Xcode Command Line Tools（xcode-select --install）                                                                                                                                                  
                                                                                                                                                                                                        
  使用步骤：                                                                                                                                                                                            
                                                                                                                                                                                                        
  # 1. 首次：创建本地签名证书（可选但推荐，否则每次重编译权限会丢）                                                                                                                                     
  make setup-signing                                                                                                                                                                                    
                                                                                                                                                                                                        
  # 2. 构建并运行                                                                                                                                                                                       
  make run                                                                                                                                                                                              
                                                                                                                                                                                                        
  # 3. 首次运行后在系统设置中授权辅助功能权限                                                                                                                                                           
                                                                                                                                                                                                        
  项目本身需要什么：                                                                                                                                                                                    
                                                                                 
  ┌───────────────┬───────────────────────────────────────────────────────────────────────────────────┐                                                                                                 
  │     项目       │                                       状态                                        │                                                                                                 
  ├───────────────┼───────────────────────────────────────────────────────────────────────────────────┤                                                                                                 
  │ 外部依赖       │ 无，纯 SPM 项目                                                                     │                                                                                                 
  ├───────────────┼───────────────────────────────────────────────────────────────────────────────────┤
  │ .build 文件夹  │ 不需要，删掉就行，swift build 会自动重新生成                                           │                                                                                                 
  ├───────────────┼───────────────────────────────────────────────────────────────────────────────────┤                                                                                                 
  │ 签名           │ Makefile 自带 fallback——没有 DoubaoLookupDev 证书会自动走 ad-hoc 签名，不影响使用       │                                                                                                 
  └───────────────┴───────────────────────────────────────────────────────────────────────────────────┘                                                                                                 
                                                                                 
  唯一需要注意的： 如果不用 make setup-signing 创建持久证书，每次重新编译后代码签名会变，macOS 可能会重置辅助功能权限，需要重新去系统设置里勾选。跑一次 make setup-signing 就能避免这个问题。
